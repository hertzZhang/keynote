# PowerPoint

日常PPT文件分享。

# Usage

- Installation Tool
```bash
npm install -g reveal-md
```
or
```bash
yarn global add reveal-md
```

- Clone file
```bash
git clone https://github.com/hertzZhang/powerpoint.git
```

- Enter the folder
```bash
cd powerpoint
```

- Play powerpoint
```bash
reveal-md ***.md
```

## Print to PDF
```bash
reveal-md ***.md --print ***.pdf
```
